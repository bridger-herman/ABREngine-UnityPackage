    Class ABRConfig

    The ABRConfig class provides access throughtout ABR to everything that is loaded in from a user's / developer's ABRConfig.json file. This is mainly useful for modifying the behaviour of the ABREngine internally, but can be occasionally helpful in other situations - for instance, if a developer needs to get access to the default bounding box / container that ABR is using.
